# 🌱 AgroScan

AgroScan es una plataforma web que utiliza Inteligencia Artificial para ayudar en la detección de posibles enfermedades en cultivos mediante el análisis de imágenes.

> **Importante:** AgroScan es una herramienta de apoyo al diagnóstico y no reemplaza la evaluación de un ingeniero agrónomo.

---

## ✨ Características

- 📸 Análisis de imágenes agrícolas mediante IA.
- 🌿 Detección de posibles enfermedades.
- 📊 Dashboard con estadísticas visuales.
- 📜 Historial de análisis.
- 📈 Nivel de confianza del diagnóstico.
- 🚫 Validación de imágenes no agrícolas.
- 🔍 Explicación del diagnóstico generado por la IA.

---

## 🚀 Tecnologías utilizadas

- Python
- Flask
- HTML5

- JavaScript

- Inteligencia Artificial (LLM Vision)

## 💡 Visión del proyecto

Actualmente AgroScan analiza imágenes tomadas desde un teléfono móvil.

En futuras versiones se planea integrar:

- 🚁 Drones
- 📷 Cámaras inteligentes (IoT)
- 🤖 Robots agrícolas
- 🛰️ Imágenes satelitales

Con el objetivo de facilitar el monitoreo de grandes extensiones agrícolas.

## ⚙️ Instalación

Clonar el repositorio

```bash
git clone https://github.com/Juancit015/AgroScan.git
```

Entrar al proyecto

```bash
cd AgroScan
```

Instalar dependencias

```bash
pip install -r requirements.txt
```

Ejecutar

```bash
python app.py
```
